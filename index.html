Parallax Scrolling

Here is an example of parallax scrolling. You can see the background scroll at a different speed than the foreground. This project helped me understand background image placement by using the long hand and short hand versions of the background properties. I also explored CSS specificity, z-index, bootstraps data-speed attribute and the proper use of the !important tag.

Put any text that you want here.

Marquee

This is a marquee that uses list items with nth-child animation delay to frame content on your page. I followed a tutorial at TeamTreehouse and this project helped me understand the CSS position property and practice the CSS fundamental of specificity. Also, I've used media queries to optimize this site for mobile devices.

Collapsible

There is a lot of built in functionallity with frameworks like Bootstrap. One such tool is the simple collapsible. During this project I learned about the variety of bootstrap buttons and animations that are availible. I also learned about the bootstrap grid layout.

You can expand your content here. This tool is simple, yet effective for saving space and keeping your website clean. It can be used instead of the accordian, which requires multiple sections of content to sift through. This collapsible requires only some content and you can close this after opening it.

Calc()

The calc() function dynamically removes the padding from the sides of this image so it fits well with the text at all screen sizes. This project helped me better understand the box model and responsive design.
